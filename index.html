R. Morsoleto

Eu crio pipelines de dados prontos para produção, simplificando processos complexos. Projetei e implementei o EtlSUS, uma ferramenta que utiliza arquivos YAML para transformar dados complexos da saúde pública brasileira em informações prontas para análise, tornando o acesso a dados de saúde mais eficiente para pesquisadores e analistas.

Construo protótipos de forma ágil, com um processo de iteração contínuo baseado em feedback. Minha abordagem, com pull requests pequenas, facilita o desenvolvimento e a revisão do código. Utilizo CI/CD, controle de versionamento e contêineres para garantir entregas de alta qualidade.

Minha prioridade é simplificar o complexo. Eu transformo projetos confusos em fluxos de trabalho claros e diretos.
